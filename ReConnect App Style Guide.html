1. Scope

ReConnect Better is a social product designed to help users discover and connect through local events. This guide covers the current visual system, key UI components, major screen patterns, and implementation notes for web and mobile teams.

Category Card

Interactive category selector used on browse screens and filter-based flows.

  • Behavior: tap to filter events with visible selected feedback.

Map View

Map-based browsing surface for events by geography and proximity.

  • Open question: no empty map, location permission, or clustering rules.

  • Open question: priority and fallback rules are not documented.

5. Screen Patterns

Pattern Purpose Required blocks Open questions
Start Page Introduce the product and provide fast entry into discovery. Status bar, header, hero section, category grid, event carousel, footer. Entry priority and personalization logic not documented.
Search Page — List View Help users review results in a structured list. Search field, filter bar, sorting dropdown, event list, map toggle, footer. No empty-result, loading, or network state documented.
Search Page — Map View Help users discover events by location and proximity. Search field, filter bar, map component, list toggle, footer. No pin clustering or permission fallback documented.
Event Page Support booking decisions with complete event information. Hero image, title, organizer, share, date/time, location, map link, details, info row, description, CTA. No documented rule for sold-out, unavailable, or expired events.
3-Column Category Grid Display category cards in a browsable grid. Three columns, wrapped rows, category icons and labels. Responsive scaling beyond base layout not documented.
Horizontal Event Carousel Enable quick event browsing from the start page. Small cards in a horizontal row with visible overflow. No pagination, snapping, or accessibility rule documented.
List / Map Toggle Switch between geographic and list-based discovery. Toggle action with “Show Map” / “Show List” logic. Preference persistence not documented.

6. Handoff Notes

iOS: Notes mention UICollectionView for category grids, UITableView for event lists, SF Pro for native feel, and support for dark mode and dynamic type sizing.

Android: Notes mention RecyclerView, GridLayoutManager, Material Design 3, Jetpack Compose, and support for system dark theme.

Cross-platform: Notes mention responsive design from 320px and up, lazy-loaded images, paginated lists, and WCAG AA compliance.

Touch targets: Minimum touch target guidance is 44px for iOS and 48dp for Android.
